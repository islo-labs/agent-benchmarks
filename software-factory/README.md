# Software Factory Grader

Vite + React app served at `/software-factory/` on agent-benchmarks.com.

## Local development

```bash
cd software-factory
npm install
npm run dev
```

Open http://localhost:5173/software-factory/

## Production build

Amplify builds this automatically on deploy via the repo-root `amplify.yml`.

To build locally:

```bash
npm run build
npm run preview
```
